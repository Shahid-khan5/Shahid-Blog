## Learn C++ on Your Phone

A full-featured C++ IDE built for students — write, compile, run, and debug C++ right on your Android device. No setup, no cloud dependency.

### What You Can Do

- **Write code** with syntax highlighting and IntelliSense
- **Compile & run** with one tap — clang is built in
- **Debug** with breakpoints and step-through
- **Practice** with guided exercises
- **Chat with Shahid Khan** directly from the editor when you're stuck
- **Upload your progress** so your work is never lost

2. If prompted, allow installation from unknown sources
3. Open the APK to install
4. Sign up with your name and email
5. Browse exercises and start coding
